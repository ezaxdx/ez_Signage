## 1. 관리자 페이지 (`/admin`) — 운영 대시보드 + AI 관리

관리자 전용. 운영 현황 중심. **운영 KPI ↔ 전체 프로젝트 현황 통합** (사용자 피드백 반영).

### 1) 운영 KPI (상단 카드)

- 진행 중 프로젝트 수 (stage='in_progress' 누적)
- 이번 주 신규 프로젝트 수
- 이번 주 발주 완료 수 (finalized_at 기준)
- 발주서 완료율 (%) = finalized_items / total_items
- 추천 다운로드 전환율 (%) = 추천 → finalized 경로 비율
- AI 추천 정확도 (rolling, 목표 70%) — 신호등 색상

### 2) 유저 관리

- 활동 사용자 목록
- 권한 단계: 일반 사용자 / 관리자 (사내 SSO 연동 시 자동 매핑)
- ⚠️ **데이터허브 연동 결정 전 — 구현 보존 + 접근 차단**
  - 사용자 퍼널 분석 영역으로 활용 검토 중

### 3) 전체 프로젝트 현황 (운영 KPI와 통합)

- 필터: 파트 · PM · 행사장 · 기간 · 상태
- 테이블 16컬럼: NO · 파트 · PM · 행사 · 상태 · 발주일 · 컨펌율 · 항목수 · AI정확도 · 담당자 · 비고 · 편집 · 다운로드 · 승인 · 반려 · 보기
- 상태별 Stacked Bar (파트 × 진행·완료·반려)
- 이번 달 일정 (D-14 ~ 오늘 ~ D+7 캘린더 형태)

### 4) AI 관리 (신규 — 사용자 추가)

> 관리자가 AI 환경을 설정할 수 있는 영역

- Gemini API 호출 건수 · 토큰 사용량 · 일일/월간 누계
- 비용 추정 ($, ₩) · 한도 알림 (예산 임계값)
- 이상 사용자 알림 (예: 동일 항목 5회 이상 재호출)
- AI 환경 설정 (모델·프롬프트·임계값 등)

## 2. 데이터 학습 관리자 (`/admin/learning`) — 학습 + 마스터 데이터 통합

환경장식물 종류·동의어 등 마스터 데이터 이쪽으로 통합.

### 1) 상단 KPI 카드

- 행사장 — 학습된 행사장 누적 수 (venues 테이블 기준)
- 환경장식물 — 표준 종류 누적 수 (11종 + 신규)
- 동의어 — 매핑된 동의어 누적 수 (비표준 → 표준)

### 2) 개요

- 학습 누적 그래프 (월별·행사장별 추가 추이)
- 단계별 분포 (입력 10% / 중간 30% / 컨펌 70% / 완료 100%) 비율
- AI 추천 정확도 추이 (월 단위 rolling)

### 3) 행사장별 학습 현황

- 행사장 · 프로젝트 수 · 항목 수 · 단계별 분포 · 정확도 추정
- 정확도 40% 이하 행사장 강조 (학습 강화 대상)
- 행 클릭 시 해당 행사장 상세 (학습 데이터 · 시설 가이드 · 예외 패턴)

### 4) 행사장 (마스터 관리)

- 행사장 추가 폼 (이름 · 지역 · 종류 · 홀 분리 · 면적 · 천장고 · 도면 업로드)
- 신규 요청 대기 (사용자 신청 → 관리자 승인·반려)
- 도면 학습 큐 (Vision API 분석 상태: queued · processing · done · failed)
- 시설 가이드 (행사장별 카테고리 제약 · 주의사항 · 완성도)
- 예외 패턴 모니터 (venue+rule 3회 이상 위반 = 가이드 검토 필요 플래그)

### 5) 동의어 (마스터 관리)

- 비표준명 → 표준명 매핑 표 (검색 · 추가 · 삭제)
  - 예: '스프링배너' = 'X배너' / '롤업' = '롤업배너' 등
- 카테고리 권장 (행사 유형별 환경장식물 기본 세트)
- 환경장식물 종류 관리 (표준 종류 · 규격 · 재질 · 레이아웃)
