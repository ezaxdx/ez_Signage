## 5. PDF / PPT 자동화 경제성 분석

사용자 질문 ("발주엑셀 외 PDF·PPT 자동 취합 경제적인가?") 에 대한 답:

### 5-1. 자료 형태별 자동화 난이도 / 비용

| 자료 형태 | 자동화 난이도 | 비용 | 경제성 | 추천 단계 |
|---|---|---|---|---|
| 발주엑셀 (xlsx) | 낮음 (표 파싱) | 0원 | ★★★ | 1단계 (현재) |
| 시안 파일명 메타 (`가로등배너_600x1800.jpg`) | 낮음 (정규식) | 0원 | ★★★ | 1단계 |
| 결과보고서 PDF (텍스트 임베드) | 중 (한국어 폰트 깨짐) | 0~소액 | ★★ | 2단계 |
| 매뉴얼 PDF (한국어 폰트 깨짐) | 중 (Tesseract OCR) | 소액 | ★★ | 2단계 |
| HWP 결과보고서 | 중-상 (한컴 API 또는 변환) | 중 | ★ | 2단계 |
| 시안 PPT 본문 | 상 (PPT 파싱) | 중 | ★ | 2단계 |
| 시안 이미지 본문 (jpg/png) | 매우 상 (Gemini Vision) | 큼 (1건당 0.001~0.01달러) | ★ | 3단계 |

### 5-2. 비용 추정 (시안 이미지 Vision API)

- 본 폴더 시안 이미지 약 1,000건 추정
- Gemini Vision 1건당 약 0.001~0.005달러
- 전체: **$1~$5** (한 번 추출 시)
- 누적 학습 시 매 행사당 추가 비용 소액

→ **결론**: 시안 Vision 비용 자체는 큰 문제 아님 (5달러 이내). 다만 추출 정확도가 충분히 높지 않을 수 있어 사람 검수가 추가 비용임.

### 5-3. 권장 우선순위

1. **1단계 (완료)**: 발주엑셀 14개 자동 파싱 + 시안 파일명 메타 추출
2. **2단계 (즉시 가능)**: 결과보고서 PDF·매뉴얼 PDF 한국어 OCR (Tesseract, 무료)
3. **3단계 (검토)**: 시안 PPT 본문 파싱 (PPT 텍스트만, 이미지 제외)
4. **4단계 (보류)**: 시안 이미지 Vision API → 비용보다 학습 가치가 큰 경우만 (예: 천정배너 정보 부재 행사장)
