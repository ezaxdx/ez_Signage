2. 엑셀 그리드 사용법

① 행 추가 — 환경장식물 종류 선택

그리드 상단 액션바의 "+ 행 추가" 버튼 클릭 → 드롭다운 표시.

  • 표준 12종 (X-배너 · I-배너 · 가로등 배너 · 가로/세로 현수막 · 통천 · 포디움 · 폼보드 · A4 가로/세로 · A3 가로/세로) 중 선택
  • 선택 시 카테고리·규격(mm)·재질이 자동 입력
  • 레이아웃 DNA 자동 적용 — Vision 분석된 6종(X배너·세로/가로 현수막·통천·포디움·폼보드)은 슬롯 위치까지 자동 배치
  • 맨 아래 "빈 행 추가 (직접 입력)"는 모든 필드 빈 채로 시작

② 행 이동

  • 각 행 좌측 ⋮⋮ 핸들을 마우스로 잡아 다른 행 위치로 드래그
  • 드래그 중 인디고색 위쪽 보더로 드롭 위치 표시
  • 순서는 localStorage에 프로젝트별 저장 → 새로고침해도 유지

③ 행 삭제

  • 각 행 우측에 호버하면 휴지통 아이콘 활성화
  • 클릭 → 확인 다이얼로그 → 확인 시 DB의 design_items + 관련 슬롯·이미지까지 함께 삭제

④ 컬럼 관리 (열 편집)

그리드 상단 "컬럼 관리" 클릭 → 패널 열림.

  • 👁️ 표시/숨김: 일시적으로 컬럼 숨김 (데이터는 유지)
  • 🗑️ 삭제: 컬럼 자체를 그리드에서 제거 (표준·커스텀 모두 가능). "리셋" 버튼으로 표준 13컬럼 복원.
  • + 추가: 패널 하단 입력란에 컬럼명 (예: "디자인업체", "출력업체", "설치시간") → "추가"
  • 순서 변경: 그리드 헤더를 직접 드래그해 다른 헤더 위에 드롭

⑤ 셀 편집

  • 더블클릭: NO·파트·장소·사용목적·품목·언어·규격(mm)·재질·수량·커스텀 컬럼 모두 편집 가능
  • Enter: 저장 / Esc: 취소
  • 규격(mm): 600×1800 또는 600x1800 형식 입력 — 자동 파싱
  • 언어: 드롭다운 (KOR / EN / EN/KOR)

3. 캔버스 동작 변경

1차 출시 캔버스 정책:
· 빈 슬롯의 📝 내용 입력 안내 텍스트 → 표시 안 함
· 빈 슬롯의 amber 배경 → 투명
· 기존에 입력된 슬롯만 캔버스에 보임
· 슬롯 위치는 내부적으로 유지되며, EditorGrid에서 텍스트 입력 시 즉시 캔버스에 등장

4. 엑셀 내보내기 동기화

"Excel 내보내기" 버튼 → 사용자가 보고 있는 그리드 상태가 그대로 엑셀에 반영됩니다.

동기화 항목동작
표시 컬럼만 출력"숨김"으로 OFF한 컬럼은 엑셀에서도 제외
컬럼 순서사용자가 드래그해 정한 순서로 출력
커스텀 컬럼"디자인업체"·"출력업체" 등 추가한 컬럼·값 모두 포함
행 순서드래그로 변경한 행 순서 그대로 출력
좌상단 / 우상단"환경 제작물 (행사명)" + 행사 로고 자동 병합 유지

폴백: localStorage에 컬럼 상태가 없으면 기존 17컬럼 표준 형식으로 출력 (기본 동작 유지).

PDF·PPT는 추후 — PDF는 인쇄용 시안 자체를 출력하는 형식이라 컬럼 개념이 다름. PPT 14컬럼 메타 테이블 동기화는 별도 작업 예정.
