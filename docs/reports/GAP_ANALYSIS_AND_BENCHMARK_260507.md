## 3. 타 SaaS 벤치마크 (유사 프로그램 조사)

### 비교 대상 — 동급 SaaS

| 도구 | 특징 | 본 프로젝트 적용 후보 |
|---|---|---|
| **Linear** | 이슈 트래커 + 누락 알림 | 누락 항목 알림 ✅ 적용 |
| **Notion** | 템플릿 갤러리·DB 뷰 | 프로젝트 복제 (next 후보) |
| **Figma** | 컴포넌트 라이브러리·자동완성 | 환경장식물 종류 카드형 미리보기 |
| **Asana** | 프로젝트 템플릿·자주 쓰는 항목 | 자주 쓰는 발주처/행사장 빠른 선택 |
| **Smartsheet** | 행사 기획 템플릿 | 표준 17컬럼 (이미 구현) |
| **Monday** | Activity Feed | 협업자 변경 알림 (next 후보) |
| **Eventbrite** | 행사 계획 단계 시각화 | stage 진행 단계 ✅ 적용 |
| **Confluence** | 인라인 헬프·Hover Tooltip | 환경장식물 종류 인라인 설명 |

### 우선 적용 후보 (자동화 4필터 통과)
