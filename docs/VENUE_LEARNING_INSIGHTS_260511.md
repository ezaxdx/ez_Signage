## 5. AI 시험 4지표 결과

| 지표 | 합격 기준 | 결과 | 판정 |
|---|---|---|---|
| 수량 정확도 | ±20% | 총량 -20% (49→39) | 🟡 경계 |
| 위치 매칭률 | 60% 이상 | 81% (17개 중 14 일치) | ✅ 통과 |
| 행사장 규칙 반영 | 1건 이상 | 4건 + 매뉴얼 보강 | ✅ 통과 |
| 환각·오답 | 0건 | 0건 (지어낸 정보 없음) | ✅ 통과 |

### 5-1. 정답지와 일치한 항목 (39개)

- 세로현수막(가로등) 18개 모두
- 가로현수막(외부) 2개
- 통천현수막 2개 (외벽 + 무대)
- X배너(실내) 15개
- 물통배너(셔틀) 2개

### 5-2. 정답지와 누락된 항목 (10개)

- 천정배너 (전시장 메인 7개 + 라운지 1개 + 컨퍼런스장 2개)
- 이유: 학습 데이터·매뉴얼 본문 모두 천정배너 정보 부재

### 5-3. 정직성 강점

학습 안 된 카테고리는 추천 자체를 안 함 = 환각 0건.
사람 검수에서 매뉴얼로 보강하면 됨 — 자동화 4필터 ④번 충족.

## 6. 행사장별 학습 데이터 수집 권고

### 6-1. 행사장 학습 우선순위

| 우선순위 | 행사장 | 사유 |
|---|---|---|
| ★ 1순위 | 킨텍스 1전시장 5홀 | 정답지 보유 + 천정배너 보강 시급 |
| 2순위 | 킨텍스 1전시장 1~4홀 / 2전시장 6~10홀 | 매뉴얼 표준 확보됨 |
| 3순위 | 코엑스 / 송도 / DCC | 대형 행사장, 도면 일부 보유 |
| 4순위 | 소노캄호텔 그랜드볼룸 등 부속 시설 | 호텔 부수 행사 패턴 |

### 6-2. 카테고리별 필수 학습 항목 (모든 행사장 공통)

| 카테고리 | 학습할 정보 | 추출 출처 |
|---|---|---|
| 외벽 광고 | 표준 규격 | 매뉴얼 |
| 외부 게이트 광고 | 게이트별 폭·광고면 | 매뉴얼 |
| 가로등배너 | 표준 규격·조 단위 | 실 발주 + 시안 |
| X배너·물통배너 | 표준 규격·실내 동선 패턴 | 실 발주 |
| 천정배너 | 규격·행잉 위치·하중 | 매뉴얼 + 실 발주 |
| 부속 시설 사인물 | 라운지·컨퍼런스장·VIP룸 패턴 | 실 발주 + 결과보고서 |

### 6-3. 자료 파싱 자동화 필요

| 자료 | 현재 상태 | 자동화 필요 |
|---|---|---|
| 발주엑셀 | xlsx 파싱 가능 (`scripts/parse_signage_lists_v4.mjs`) | ✅ 구현됨 |
| 시안 파일명 | 수동 분류 | 메타 자동 추출 스크립트 |
| 결과보고서 HWP | 미파싱 | 한컴 API 또는 변환 |
| 매뉴얼 PDF (한국어) | 부분 실패 (폰트 깨짐) | Tesseract 한국어 OCR |
| 결과보고서 PDF | 미파싱 | pdftotext + 한국어 OCR 백업 |

### 6-4. 시스템 동작 보강 원칙

1. **모르는 카테고리는 추천 안 함**: 학습 데이터에 없으면 "추천 없음 + 매뉴얼 보강 권고" 자동 표기 → 환각 방지
2. **행사 격에 따른 보정 룰**: 국제·VIP·참가자 수 등 메타 정보로 표준 규격 보정 (예: WSCE 2021 600×1800 vs 스마트국토 600×2000)
3. **부속 시설 인지**: 도면에 표기된 부속 시설(라운지·컨퍼런스장 등)을 자동 인지하는 휴리스틱
4. **사람 검수 1회 필수**: 자동화 4필터 ④번 — 매번 검토 아닌 1회 검수 후 통과 형태
