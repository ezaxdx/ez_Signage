# batch-scan-events — 행사별 폴더 일괄 스캔

## 목적

`참고자료/환경장식물 행사별/` 내 **모든 폴더**를 순회하여
엑셀/파일명에서 추출 가능한 데이터를 한 번에 수집한다.

이미지 AI 분석은 포함하지 않음 (별도 `/analyze-event-folder` 사용).

---

## 실행 시 수행할 작업

1. **폴더 목록 열거** — 모든 하위 폴더 이름 파악
2. **파일 확장자별 집계**:
   - `.xlsx` / `.xls` → 발주서 후보 (컬럼 구조 파악)
   - `.jpg` / `.png` → 이미지 샘플 수량
   - `.pdf` / `.pptx` → 디자인 가이드 후보
3. **파일명 패턴 분석** → 제작물 종류명 추출 후 동의어 후보 목록 생성
4. **수행실적 엑셀 대조** — `Ezpmp_수행실적리스트_20260506.xlsx` 로딩 후
   각 폴더명과 매칭되는 행 수집
5. **결과 요약 출력**:
   ```
   폴더명 | 엑셀파일 | 이미지수 | 행사명(수행실적) | 연도 | 비고
   ```

---

## 분석 결과 반영 위치

- 행사 이력 데이터 → `supabase/migration_v5_data_tables.sql`의 `event_history` INSERT
- 신규 동의어 → `signage_synonyms` INSERT
- 행사장 정보 → `venue_info` INSERT
- 이미지 샘플 목록 → 향후 AI 분석 큐 (`scripts/analysis_queue.txt` 신규 생성)

---

## 주의

- `[미분류]` 폴더는 별도 플래그 처리 (수동 검토 필요)
- `브라질 벨렘` / `프랑스 파리` → `region: '해외'` 로 venue_info 추가
- 같은 행사장 이름이 여러 폴더에 있으면 venue_info는 1개만 (UPSERT)
