# /weekly-report

지난 7일 commits·변경 라인·D-day 카운트 자동 추출하여 주간 보고 양식 생성.

## 실행 절차

1. `npm run report:weekly` 실행 → `docs/weekly_report_YYMMDD.md` 자동 생성
2. 생성된 파일 사용자에게 안내·항목 채워야 할 부분 명시 (잘 된 점·문제·다음 주 계획·시간 절감 측정)
3. 분류 = [진척 공유] 첫 줄 명시
4. docx 변환 권장 = `md_to_docx.ps1` 도구로 정 팀장·곽 이사 보고용 변환 가능

## 출력 형식

- 객관 수치 (커밋·라인 수·D-day) = 자동
- 주요 작업 commits 요약 = 자동
- 잘 된 점·문제·다음 주 계획·시간 절감 = 사용자 채움
- 자동화 미션 진척 = D 평가 회복 트랙 가시화

## 금지

- 사용자 미작성 항목 임의 작성 X
- 잘 된 점·문제 = 본인 인지 영역·자동 생성 X
- "97%·완벽" 표현 X
